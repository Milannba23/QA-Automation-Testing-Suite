# QA Automation Suite - KupujemProdajem

## Description
This repository contains automated UI tests for the **KupujemProdajem** platform. It demonstrates advanced web automation techniques, focusing on search functionality and user flow validation.

## QA Automation Highlights
- **Dynamic Search Testing**: Automates the process of searching for items and verifying results.
- **Selector Strategy**: Uses robust CSS Selectors and XPaths to interact with complex web elements.
- **Error Handling**: Implements wait strategies (Explicit Waits) to handle asynchronous page loading.
- **Reporting**: Provides clear console output for test pass/fail status.

## Tech Stack
- **Language**: Python 3.x
- **Framework**: Selenium WebDriver / PyTest
- **Tools**: Chrome WebDriver
